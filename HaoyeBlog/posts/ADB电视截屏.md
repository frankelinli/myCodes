📺 硬核教程：如何使用 ADB 命令获取智能电视截屏？

想把电视上精彩的画面保存下来？这就是你在寻找的技术流方案。

作为一名开发者或数码爱好者，我们经常需要对智能电视（Android TV）进行操作，比如截取当前屏幕画面用于分享、测试或记录这是瞬间。虽然部分电视遥控器有截图键，但**将图片从电视传输到电脑**通常非常繁琐。

今天，我来教大家如何使用 **ADB (Android Debug Bridge)** 工具，通过几行命令轻松实现**无线截屏**并自动保存到电脑！

![image-20260109204342797](https://haoyelaiga.com/wp-content/uploads/2026/01/image-20260109204342797.webp)

---

## 🛠️ 准备工作

在开始之前，请确保：

1. **开启 ADB 调试**：在电视的【设置】->【关于】中找到版本号，连续点击 7 次开启开发者模式，然后进入开发者选项开启 **“USB调试”**。

   :::tip

   不同电视开启ADB的方法不一样

   :::

2. **网络互通**：确保电脑和电视连接在同一个 WiFi 网络下。

3. **获取 IP 地址**：在电视的网络设置中查看 IP 地址（例如 `192.168.21.128`）。

4. **电脑端 ADB 环境**：下载 Platform-Tools 。解压缩即可，不需要安装。

   记得ADB命令需要在Platform-Tools目录下进行。

---

## 💻 第一步：连接电视

打开电脑的终端（CMD 或 PowerShell），输入以下命令连接电视：

```bash
adb connect 192.168.11.128
```

*（请将 IP 替换为你实际的电视 IP）*

如果看到 `connected to 192.168.11.128:5555`，恭喜你，连接成功！

---

## 📸 第二步：核心截屏命令

### 🚀 快速操作：单条命令截图

为了最高效地将电视屏幕截图直接保存到你**当前电脑终端所在的目录**，最推荐的方法是使用这条命令：

```
adb exec-out screencap -p > screenshot.png
```

或者以下分步骤的命令：

```bash
adb shell screencap /sdcard/screen.png —— 截图

adb pull /sdcard/screen.png —— 下载截图
-------------------------------------------------

adb shell screenrecord /sdcard/video.mp4 —— 录屏
				[CTRL+C 结束录屏]
				
adb pull /sdcard/video.mp4 —— 下载录像
```



:::danger :fire:

我觉得酒店的电视里很多肯定开启了USB或者无线ADB功能，特别是无线ADB，那么你在看的什么电视，电视屏幕显示的什么内容，都会被管理员很容易的远程截屏！！

:::
