**LocalSend 简介**

LocalSend 是一个开源的点对点文件传输工具，支持跨设备、跨平台（Windows、macOS、Linux、Android 等）快速分享文件、图片、链接等，无需互联网、无需注册。它通过本地局域网（Wi-Fi/热点）直连，传输速度快且安全。类似 AirDrop，但不限苹果生态，适合 Android ↔ 电脑、不同系统间互传。界面简洁，操作简单：打开应用→选择文件→对方扫码或连接即可传送。对比传统蓝牙或网盘，LocalSend 更省时、省流量，适合临时分享大文件。



## LocalSend 是什么？

LocalSend 是一款**开源、免费的局域网文件传输工具**。
 一句话概括：**在同一个网络里，设备之间直接“面对面”传文件**。

它支持的平台包括：

- Windows
- macOS
- Linux
- Android
- iOS

也就是说：
 👉 安卓 ↔ Windows
 👉 iPhone ↔ Linux
 👉 公司电脑 ↔ 私人手机
 都能用，同一个套路解决。

------

## 它是怎么传文件的？

LocalSend 不经过云端，也不需要登录账号，而是：

1. 设备连接到**同一个局域网**（Wi-Fi / 热点）
2. 自动发现附近设备
3. 点选 → 确认 → 传输完成

:::note 太好用了！:airplane:

整个过程像在“隔空递 U 盘”，而不是“先上传、再下载”。

:::

顺便一提：
 **局域网速度有多快，它就能跑多快**，传视频、大文件尤其明显。
