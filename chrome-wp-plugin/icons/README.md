# WordPress 会员验证插件 - 图标说明

本文件夹包含插件使用的图标文件。

## 图标规格

- **icon16.png**: 16x16px - 扩展管理页面和上下文菜单
- **icon32.png**: 32x32px - Windows 计算机设置页面
- **icon48.png**: 48x48px - 扩展管理页面
- **icon128.png**: 128x128px - Chrome 网上应用店和安装对话框

## 设计要求

- 使用 PNG 格式
- 透明背景
- 主色调：蓝色渐变 (#4facfe 到 #00f2fe)
- 图标风格：现代、简洁、专业
- 包含 WordPress 或会员相关元素

## 临时方案

由于当前没有实际的图标文件，你可以：

1. **使用在线图标生成器**创建图标
2. **下载免费图标**并进行自定义
3. **委托设计师**创建专业图标

推荐网站：
- https://www.iconfinder.com/
- https://iconify.design/
- https://www.flaticon.com/

## 图标创建完成后

将对应尺寸的 PNG 文件放置在此文件夹中：

```
icons/
├── icon16.png
├── icon32.png
├── icon48.png
└── icon128.png
```
