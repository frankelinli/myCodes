title: ICS Code of Conduct 社会责任行为准则
---

每个 ICS（即社会条款行动）的成员–已经建立了属于他们各自的行为守则。ICS 条款是基于
ILO（国际劳工组织）公约建议，同时也基于《世界人权宣言》基础上建立的(详细请看附录中列表)。通过签订此项行为守则，供应商和它所拥有的工厂、合作伙伴应当遵守约定的协议，尊重条款并且肩负起相应的责任。
