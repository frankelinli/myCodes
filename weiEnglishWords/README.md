# English Collector（Chrome 插件）

一个轻量级英语学习插件：
- 在网页中选中文本，右键即可一键收藏
- 在弹窗中搜索、筛选、收藏、删除，支持随机复习
- 可为每条英语内容填写中文翻译，方便英中对照学习
- 支持导出 JSON 与打印导出

## 安装使用

1. 打开 Chrome，进入 `chrome://extensions/`
2. 打开右上角“开发者模式”
3. 点击“加载已解压的扩展程序”，选择当前项目目录
4. 打开任意网页，选中文本后右键使用：
  - `收藏到英语剪藏`
5. 点击插件图标进入弹窗管理收藏内容

## 功能说明

- **网页采集**：基于右键菜单快速收藏
- **复习整理**：搜索、仅看收藏、随机复习
- **翻译补充**：在每条收藏下填写并保存中文翻译
- **导出**：
  - `导出 JSON`：保存完整数据做备份
  - `打印导出`：生成打印页并调用浏览器打印

## 数据存储

- 所有数据保存在浏览器本地 `chrome.storage.local`
- 卸载扩展会清除本地数据，建议定期导出 JSON 备份
