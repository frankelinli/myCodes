一键推送网址到百度搜索引擎

```
@echo off

curl -H "Content-Type:text/plain" --data-binary @urls.txt "http://data.zz.baidu.com/urls?site=https://dfadwiki.com&token=VOIWWefdedMTPIXSnZDh"

pause
```

这是我自己写的命令行代码，把URL连接放到urls.txt文档里，然后命令行里运行上面代码，就可以推送URL到百度搜索引擎。
