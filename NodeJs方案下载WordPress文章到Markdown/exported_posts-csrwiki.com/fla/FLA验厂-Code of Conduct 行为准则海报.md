美国公平劳动协会（FLA）的《公平劳动规范》是一个针对工厂和农场的标准，旨在维护工人的权利。该规范基于国际劳工组织和国际公认的人权标准。FLA的成员公司同意在其全球供应链中遵守这些工作场所标准，以保护工人并改善工厂和农场的工作环境。
