标题：amfori BSCI 认可第三方审核公司名单**

amfori BSCI（商界社会责任倡议）审核必须由经过 amfori 认可并具备相应资质的独立第三方审核公司执行。这些审核公司需要成功通过 amfori 的认可程序，以确保其审核员完全理解 amfori BSCI 行为准则并能以一致和可靠的方式实施审核。

您可以通过以下链接访问 amfori 官方提供的认可审核公司完整名单：

**amfori 官方认可监控公司列表 (Official amfori Approved List of Monitoring Companies)**

- **链接：** https://www.amfori.org/content/approved-monitoring-companies

**重要提示：**

1. **官方来源：** 上述 amfori 官方链接是获取最新、最准确认可审核公司名单的唯一可靠来源。该列表会定期更新。
2. **本网站声明：** [csrwiki.com](https://csrwiki.com/) 本页面明确指出：“请注意：我们不列出审核公司。请参考 amfori 网站获取认可的审核公司列表。” (“Please note: We don’t list audit companies. Please refer to the amfori website for the list of approved audit companies.”)
3. **自行验证：** 在选择审核公司时，请务必直接通过 amfori 官方名单进行核实，以确保其认可资格当前有效。

**关于审核公司（来自网页的通用信息）：**

- 审核公司需获得相关认证，例如符合 ISO 17020 或 ISO 17021 等标准。
- 审核员必须接受过 amfori BSCI 审核员培训并通过考试。
- 认可资格并非永久有效，审核公司需要定期接受重新评估。
